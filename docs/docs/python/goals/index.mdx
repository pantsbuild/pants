Details on the Python-related goals implemented in the Python backend.

---

- [fmt](./fmt.mdx): autoformat source code.
- [lint](./lint.mdx): lint source code in check-only mode.
- [package](./package.mdx): package your code into an asset, e.g. a wheel or a PEX file.
- [repl](./repl.mdx): open a REPL (standard shell or IPython).
- [run](./run.mdx): run an executable or script.
- [test](./test.mdx): run tests with Pytest.
- [check](./check.mdx): run MyPy.
